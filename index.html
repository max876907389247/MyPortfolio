FoodSource ИИ · MySQL · Node.js

FoodSource — поиск поставщиков продуктов питания

B2B-платформа для покупателей (кафе, рестораны, ритейл, кейтеринг) и поставщиков (производители, дистрибьюторы, оптовики). Сервис отвечает на вопрос: как быстро найти надёжного поставщика под свои критерии и оформить закупку или откликнуться на запрос покупателя.

Задача
Собрать MVP двустороннего маркетплейса: каталог и скоринг поставщиков, сравнение, заказы, заявки на закупку, роли покупателя, поставщика и администратора.
Моя роль
Проектирование и разработка с активным использованием ИИ: структура MySQL, API и интерфейсы; ChatGPT, Claude и Cursor — для черновиков SQL, логики сервиса, UI-текстов, отладки и ускорения рутинных этапов.
Функциональность
Покупатель: фильтры, рейтинг, сравнение до 3 поставщиков, заказ, заявки, избранное. Поставщик: каталог заявок, КП, свой каталог товаров, входящие заказы. Админ: модерация пользователей, поставщиков и заявок.
Стек
ChatGPT, Claude, Cursor · MySQL · HTML, CSS, JavaScript · Node.js · Docker.
Релевантность
Кейс AI Generalist на продуктовой задаче: ИИ ускорил путь от идеи до рабочего сервиса — проектирование, код, контент и проверка решений в одном MVP.
