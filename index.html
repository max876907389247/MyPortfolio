Соответствие роли

Почему мой опыт релевантен frontend-позиции

Требование

Вёрстка по макетам, адаптив под мобильные устройства

Мой опыт: создавал адаптированный макет сайта колледжа УПК МЦК, согласовывал его с главным разработчиком, адаптировал сайт под мобильные устройства и разработал навигацию.

Результат: удобный мобильный UX на сайте колледжа, перенос дизайна в рабочую вёрстку без потери структуры.

Требование

Уверенное владение HTML, CSS, JavaScript

Мой опыт: в «Альянс АйТи Технолоджи» готовил Word-документы с преобразованием в HTML с использованием HTML, CSS и PHP — от 370 до 500 документов в месяц. Преподаю основы web-разработки и JS студентам.

Результат: стабильный поток production-ready HTML-разметки в сжатые сроки.

Требование

Работа с базами данных (MySQL / SQL)

Мой опыт: работал с MySQL при создании сайтов во время учёбы и в самостоятельных проектах. На сайте «DrillWood» выводил каждую категорию товаров на отдельную страницу, делал карточку товара с размером, цветом и расчётом цены, а также форму заказа с сохранением данных клиента.

Результат: связка frontend + PHP + MySQL: каталог, страницы товаров и приём заказов из одной базы данных.

Требование

Работа в команде, ответственность за результат

Мой опыт: в отделе разработки «Альянс АйТи» выполнял роль тимлида: распределял задачи, контролировал качество и сроки, обучал новых сотрудников. Переработал регламент постановки задач.

Результат: на 40–45% меньше уточняющих вопросов, сроки выполнения задач сократились на 23%.
