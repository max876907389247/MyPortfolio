# MyPortfolio — лендинг AI Generalist

Статический одностраничный сайт-портфолио [Максима Трофимова](https://max876907389247.github.io/MyPortfolio/). Адаптирован под вакансию **AI Generalist**: показывает практическое применение ИИ в работе, обучении и повседневных задачах.

**Демо:** https://max876907389247.github.io/MyPortfolio/

## О проекте

Лендинг на HTML, CSS и JavaScript без сборщиков. Структура заточена под оценку релевантности опыта:

- **Hero** — позиционирование и профиль под AI Generalist
- **Соответствие роли** — «требование → опыт с ИИ → результат»
- **Проекты** — УПК МЦК (учебные материалы + ИИ), Альянс АйТи (ускорение задач), DrillWood
- **Опыт** — хронология с акцентом на ИИ-инструменты
- **Контакты**
